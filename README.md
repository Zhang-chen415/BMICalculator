# BMI 計算機 (Windows Programming II 練習)

## 專案簡介
這是一個使用 C# WinForms 開發的 BMI 計算工具，具備即時計算、輸入防錯以及體位視覺化提示功能。

## 主要功能
* [cite_start]**BMI 計算**：輸入身高體重，自動計算數值並判斷體位 [cite: 442, 443]。
* [cite_start]**輸入驗證**：防呆機制，若輸入非數字或負數會跳出警告視窗 [cite: 457, 484]。
* **介面強化** (加分項)：
    * 新增清除按鈕與離開按鈕。
    * 設定 Enter 鍵為預設計算快捷鍵。
    * 根據計算結果自動變更背景色，並優化深色背景下的字體對比度。

## 開發環境
* Visual Studio 2022
* [cite_start].NET Framework 4.7.2 [cite: 15, 24]
